Программа написана с использованием Python 3.8. Для графического интерфейса используется tkinter.

Данный рендер написан мной как с целью разобраться в трассировке лучей, так и с целью применить изученные средства Python на практике. Основой для программы послужила эта статья https://habr.com/ru/post/342510/ а так же мой опыт работы в Blender 3D, и конечно я самостоятельно размышлял о том, как это могло бы работать. 

Написанный мной рендер не особо точен, как минимум из за постоянных преобразований из float в int. Так же, если какая-то компонента цвета больше 255, то она обрезается, в то время как, например в Cycles, компонента может выходить за границы для лучшей точности. Однако, цели обучения вполне достигнуты и таким кодом.
